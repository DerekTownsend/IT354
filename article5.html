Node

Node is a JavaScript runtime enviroment that allows for you to use and exceute JS scripts within a server enviroment. Plenty of many popular JS frameworks and libraries are built upon Node and they have changed the way we use JS as a whole. Uses the V8-JS engine which powers chrome and allows for us to execute our JS commands in the browser. The fact that V8 was made the core of chrome allowed for Nodes popularity to surge and revolutionize JS.

V8 was coded in C++ and has several iterations as time has passed on. The core reason being that they strive to improve browser preformance as best as possible. Having a slow browser is essentially a death sentence and as time progresses we expect faster and more effecient allocation of browser resources. With the newest iteration of the engine ES6's syntactic sugar and other features were added excluding modules.
