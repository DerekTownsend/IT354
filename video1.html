Values and Types

Numbers

Numbers are pretty much exactly what you think they are. They are integers most often used to do calculations or keep track of a count.

Infinity and -Infinity

These are unique values used to evaluate extremely large or small numbers. Infinity is the largest possible number that JS can store and -Infinity is the smallest.

NaN

NaN or Not a Number is used to categorize anything that isn't a number. This mostly is used when evaluating wether or not an input or data you recieve is a number or not.

Strings

A string is essentially made up of characters and is they way that we store text in javascript.

Unary Operators

typeof

Essentially this returns what data type you have it evaluate.

Unary plus

This is your typical addition symbol. If you use it to add 2 numbers together it will return the sum of those numbers. If you have a string with a number inside of it and place the plus in front of the string it will return the string but now as a number. It can also be used to combine strings together.

Unary Minus

This is your typical subtraction symbol. If you use it to subtract 2 numbers together it will return the difference. If you have a string with a number inside of it and place the plus in front of the string it will return the string but now as a number and negate the value.

Logical Not

An exclamation point used to negate its boolean value. If something is true it becomes false and vice versa. Most often used in loops if you want something to be done until the condition is no longer true.

Double Logical Notation

Two exclamation points used to get the absolute boolean value. It essentially makes true equal true and false equal false. The numbers 1 and 0 are truthy and falsey respectively along with some other values and data types. If these values are evaluated in a if statement or loop they will be the same as true and false. If you want to store the boolean value of one and zero you can use the Double Logical Notation.

Increment

This is two addition symbols usually before or after a variable. It will increase that variable's value by one and update the variable.

Decrement

This is two subtraction symbols usually before or after a variable. It will decrease that variable's value by one and update the variable.

Delete

Delete is used to remove an attribute from an object and will return true if the object is deleted or if it never existed. If you try to call that object's attribute that you deleted it will return undefined

Hoisting

Hoisting is when during the compilation and processing of your code it takes all of your declarations for your variables using var and named functions and moves them to the top. It is done so that you won't get any variable undefined issues, but can also cause some errors. If you have variables with the same name you might end up using the wrong values for your functions. To fix those errors we now use let and const along with function expressions to stop js from doing this.
