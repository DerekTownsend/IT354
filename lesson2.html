Lesson 2

Functions

Named Functions

Numbers are pretty much exactly what you think they are. They are integers most often used to do calculations or keep track of a count.

function add(num1, num2){
return num1 + num2;
}
console.log(add(1,2)) // 3

Function Expressions

These are unique values used to evaluate extremely large or small numbers. Infinity is the largest possible number that JS can store and -Infinity is the smallest.

const subtract = function(num1, num2){
return num1 - num2;
}
console.log(subtract(1,2)) // returns -1

Arrow Functions

NaN or Not a Number is used to categorize anything that isn't a number. This mostly is used when evaluating wether or not an input or data you recieve is a number or not.

const divide = (num1, num2) => {
return num1 / num2;
}
console.log(divide(1,2)) // returns 0.5
// can also be written
const square = num1 => {
return num1 ** 2
}
console.log(square(2)) // 4
// or
const cube = num1 => num1 ** 3
console.log(cube(2)) // 8

Other Topics

Optional Arguments

Essentially this returns what data type you have it evaluate.

function printName(firstName){
console.log(`Hello, ${firstName}!`)
}
printName("Dave",5,"forty") // "Hello, Dave!" Ignores Additional parameters and doesn't throw error
printName() // "Hello, undefined!" because it requires a parameter otherwise undefined

Rest Parameters

This is your typical addition symbol. If you use it to add 2 numbers together it will return the sum of those numbers. If you have a string with a number inside of it and place the plus in front of the string it will return the string but now as a number. It can also be used to combine strings together.

function printNames(...names){
// takes all of the parameters entered and puts them into an array
for (name of names){
console.log(`Hello, ${name}!`)
}
}
printNames("Dave","Jane", "John","Satania")
// prints:
// Hello, Dave!
// Hello, Jane!
// Hello, John!
// Hello, Satania!
// should be used at the end otherwise it takes assumes all parameters are part of the spread operator

Scope
