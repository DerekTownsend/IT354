Lesson 5

Modules

Sections of code that are highly reuseable because they preform a specific task that is needed throughout the project. Makes it so that if the code needs to be modified you only have to change it in one place rather than in every section that the code is used. Helps us remain consistent when it comes to the principle of not repeating ourselves.

Higher Order Functions

These are functions that return another function. It allows for you to modularize your code and make it more reuseable. Allows for you to reuse functions and modifiy what they do as you need them. We can now write more dynamic code.

Advanced Array Methods

Many of these exist and they let you iterate through an array and modify the the contents within and return the modified array or results.
